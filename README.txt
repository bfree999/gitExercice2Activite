# Projet Chronos

Le projet Chronos et un chronomètre écrit en Javascript pour être inséré dans des pages HTML.

# Apprentissage rapide

1 Insérez le fichier `chrono.class.js` dans votre page HTML.
2 Instanciez la classe Chrono() dans une variable de portée globale.
3 Inspirez-vous de la fonction `startstopreset()` pour gérer votre chronomètre.
